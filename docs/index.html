A simple filter that respects existing Vinted choices

EcoChineur adds selected filters to the active Vinted URL while preserving existing filters: search text, price, sort order, condition, brands, and materials.

Referenced brands

Brands are bundled only when their Vinted identifier is verified. The dev release includes Armor Lux, Faguo, and Veja.

Natural materials

Select linen, cotton, wool, silk, cashmere, alpaca, mohair, or merino to enrich a Vinted search.

Conservative merge

`brand_ids[]` and `material_ids[]` are added without removing IDs that are already present.

Private by design

EcoChineur collects no data and uses no user data for analytics, advertising, profiling, audience measurement, model training, sharing, or resale.

No backend

The extension has no server and makes no extension-owned network calls.

No telemetry

No analytics, tracking, pixels, beacons, or usage measurement.

Local preferences

Checked options and language preference stay in `chrome.storage.local` on the user's device.

Data referenced in the dev release

The bundled database is intentionally small: only entries with verified Vinted IDs are published.

Install the dev build

The current version is intended for local testing from the GitHub repository.

  1. Download or clone the GitHub repository.
  2. Run `npm install`, then `npm run build`.
  3. Open `chrome://extensions` and enable Developer mode.
  4. Click "Load unpacked" and select the `dist/` directory.
  5. Open `https://www.vinted.fr/catalog`, then open EcoChineur.
